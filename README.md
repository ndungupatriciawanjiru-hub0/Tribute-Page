# 🖥️ Ada Lovelace – Tribute Page

-----

## 📌 Project Overview

This project is a tribute web page dedicated to **Ada Lovelace** (1815–1852), widely recognised as the world’s first computer programmer.
It presents her biography, key life milestones, notable quotes, and contributions to the world of computing in a clean, structured, and visually appealing layout.

The project demonstrates the use of HTML and CSS to build a responsive, well-structured tribute page that honours a historical figure who revolutionised the IT world.

-----

## 🎯 Problem Statement

Many people are unaware of the women who laid the foundation of modern computing.
This tribute page brings Ada Lovelace’s story to life in a simple digital format, making her achievements accessible and easy to read for anyone visiting the page.

-----

## ✨ Features

- Title and subtitle identifying the tribute subject
- Responsive image with a descriptive caption
- Biography section introducing Ada Lovelace
- Timeline of key life milestones from 1815 to 2009
- Notable quotes highlighted in styled quote blocks
- External link to Wikipedia for further reading
- Fully responsive design for mobile, tablet, and desktop

-----

## 📄 Pages Included

- **Tribute Page (index.html)** – Single page with all tribute content including image, biography, timeline, quotes, and external link

-----

## 🛠️ Technologies Used

- **HTML** – Semantic page structure and required IDs
- **CSS** – Styling, layout, and responsive design using media queries

## 🚀 How to Run the Project

1. Download or clone the repository
2. Open the project folder in **VS Code**
3. Right-click `index.html` and select **Open with Live Server**
4. The tribute page opens in your browser
5. Click the Wikipedia link to read more about Ada Lovelace

-----

## 🌐 How to Host on GitHub Pages

1. Create a new GitHub repository named `tribute-page`
2. Upload `index.html` and `README.md`
3. Go to **Settings → Pages**
4. Under **Branch**, select `main` → click **Save**
5. Your live site will be at:
   https://ndungupatriciawanjiru-hub0.github.io/Tribute-Page/
